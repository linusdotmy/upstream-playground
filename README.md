# Upstream Logging Playground

Here is an example of how to use the ``upstream-sdk`` to log events to [Upstream](https://up.linus.my).

## Quick Start

Clone the repo:
```bash
git clone https://github.com/linusdotmy/upstream-playground
```

Install dependencies:
```bash
npm install
```

Configure your **Upstream API Key** in ``.env``

Run the example script:

```bash
npm run logging
```
